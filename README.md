## ‚ú® Key Features

* **Aurora UI:** A high-end, glassmorphic interface with fluid background animations and responsive panels.
* **Intent Engine:** A backend heuristic algorithm that scores subscriptions (0-100) based on value and usage risk.
* **Financial Impact Simulation:** Real-time calculation of "Current Path" vs. "SmartPath" to visualize wasted capital.
* **Money Map:** Visual breakdown of spending by category and a 12-month trend analysis.
* **Crystal Ball:** A predictive calendar view that forecasts upcoming auto-debits to prevent overdrafts.
* **One-Click Actions:** Simulated API triggers to "Cancel" or "Optimize" (negotiate better rates) subscriptions instantly.
